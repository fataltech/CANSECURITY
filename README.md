# Car Security Analyzer App

This App is used for talking to our CAN Analyzing tool. Via Bluetooth, along with recieving notifications of the car using our CAN Security Analyzer. If the CAN Security Analyzer detects an unusual change of state. It will go on to give a warning signal to the owner using WiFi. The sole purpose is for the car attached with the CAN Security Analyzer to be parked near either non-password WiFi, or configure to connect to your home network.
